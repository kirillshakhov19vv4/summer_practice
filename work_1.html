Работа 1

Структурное форматирование веб-страницы

Цель работы

Изучение элементов HTML для структурирования текста.

Задание

Разработать HTML программу для вывода на web-страницу титульного листа отчета по лабораторной работе.

Необходимые знания

  • Особенности использования языка HTML
  • Принципы структурирования текстов
  • Принципы применения стилей к тексту документа с помощью CSS

Порядок выполнения работ

  1. Изучить теорию;
  2. Разработать программу в соответствии с заданием;
  3. Составить отчёт о выполнении лабораторной работы.

Указания по выполнению работы

В качестве образца титульного листа взять титульный лист, который вы будете разрабатывать для отчета по данной лабораторной работе. В программе использовать по возможности максимальное число элементов структурирования текста. Для оформления дизайна (представления) документа использовать внедренные листы CSS.

Контрольные вопросы

  1. Какие элементы используются для выделения текста или его части?
  2. Что необходимо иметь для создания домашней страницы Web сервера?
  3. Какими командами можно изменить размер шрифта отдельных символов текста?
  4. Должны ли имена тэгов набираться строчными буквами?
  5. Для каких целей используются контейнерные элементы cite, code и address?
  6. Для каких элементов можно использовать атрибут «align="center"»?
  7. Какой принцип необходимо использовать при "открытии" и "закрытии" вложенных элементов?
