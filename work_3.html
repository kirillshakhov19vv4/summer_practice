Работа 3

Создание таблиц

Цель работы

Изучение элементов HTML для создания таблиц.

Задание

Разработать HTML программу для вывода на Web страницу таблицы успеваемости учебной группы за два семестра.

Необходимые знания

  • Особенности использования языка HTML
  • Изучить принципы создания таблиц
  • Принципы применения стилей к тексту документа с помощью CSS

Порядок выполнения работы

  1. Изучить теорию;
  2. Разработать программу в соответствии с заданием;
  3. Составить отчёт о выполнении лабораторной работы.

Указания по выполнению работы

В таблице успеваемости должны присутствовать столбцы: ФИО студента, Имя дисциплины, Оценка, Номер семестра, Номер группы. В программе использовать по возможности максимальное число атрибутов элемента создания таблиц. Для оформления дизайна (представления) документа использовать внедренные листы CSS.

Контрольные вопросы

  1. С помощью каких элементов создается таблица на Web странице?
  2. Какие существуют атрибуты элемента «table»?
  3. Как изменить цвет фона любой ячейки таблицы?
  4. Как объединить в таблице несколько ячеек одной строки или одного столбца?
  5. Как можно изменить тип шрифта текста?
  6. Как включить в исходный текст HTML комментарии?
  7. Для каких целей используется элемент «pre»
