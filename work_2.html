Работа 2

Форматирование символов и организация списков

Цель работы

Форматирование символов и организация списков.

Задание

Разработать HTML программу для вывода на web-страницу титульного листа отчета о лабораторной работе.

Необходимые знания

  • Особенности использования языка HTML
  • Принципы форматирования символов текстов
  • Организация в списки
  • Принципы применения стилей к тексту документа с помощью CSS

Порядок выполнения работы

  1. Изучить теорию;
  2. Разработать программу в соответствии с заданием;
  3. Составить отчёт о выполнении лабораторной работы.

Указания по выполнению работы

В качестве образца взять техническое задание, которое вы будете разрабатывать для отчета по данной лабораторной работе. В программе использовать по возможности максимальное число элементов форматирования символов текста. В программе должны присутствовать нумерованные и ненумерованные списки. Для оформления дизайна (представления) документа использовать внедренные листы CSS

Контрольные вопросы

  1. В чем состоят особенности языка HTML?
  2. Какие типы списков поддерживает язык HTML?
  3. Какие атрибуты можно использовать с элементами, создающими списки?
  4. Как будет выглядеть список в броузере, если элемент, определяющий его, имеет вид «ol type="A" start="4"»?
  5. Какие элементы не требуют наличия «закрывающего» элемента?
  6. Какой номер в элементе заголовка соответствует нормальному размеру шрифта?
  7. Какие элементы языка HTML и почему называются контейнерными?
