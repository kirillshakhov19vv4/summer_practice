Работа 5

Разработка документа сложной структуры

Цель работы

Изучение элементов HTML управления цветом и вставки объектов в документ.

Задание

Разработать HTML программу для вывода на Web страницу отчета о лабораторной работе.

Необходимые знания

  • Особенности использования языка HTML
  • Принципы управления цветом
  • Принципы вставки объектов
  • Принципы применения стилей к тексту документа с помощью CSS

Порядок выполнения работы

  1. Изучить теорию;
  2. Разработать программу в соответствии с заданием;
  3. Составить отчёт о выполнении лабораторной работы.

Указания по выполнению работы

Каждый раздел отчета должен представлять отдельную страницу. Переход от одного раздела к другому должен выполняться с помощью ссылок. В программе использовать цветовые эффекты. Для оформления дизайна (представления) документа использовать внедренные листы CSS.

Контрольные вопросы

  1. Как определить ссылку внутри страницы?
  2. Как определить ссылку на другую страницу и внутри другой страницы?
  3. Как определить ссылку на рисунок?
  4. Какие типы рисунков поддерживает язык HTML?
  5. Когда и в каких элементах используется отрибут alt?
  6. Какими командами можно изменить цвет фона или цвет блока текста на Web странице?
  7. Как можно использовать рисунок для указания ссылки?
