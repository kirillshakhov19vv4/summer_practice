Работа 4

Создание фреймов

Цель работы

Изучение элементов HTML для создания фреймов.

Задание

Разработать HTML программу для вывода на Web страницу фреймов с анкетными данными студента.

Необходимые знания

  • Особенности использования языка HTML
  • Изучить принципы создания фреймов

Порядок выполнения работы

  1. Изучить теорию;
  2. Разработать программу в соответствии с заданием;
  3. Составить отчёт о выполнении лабораторной работы.

Указания по выполнению работы

Выполнить создание фреймов. В них должны присутствовать поля: Фамилия студента, Имя студента, Год рождения, Адрес студента, Номер группы. В программе использовать по возможности максимальное число атрибутов элемента создания фреймов. Для оформления дизайна (представления) документа использовать внедренные листы CSS.

Контрольные вопросы

  1. Что такое фреймы в языке HTML?
  2. Какие типы фреймов поддерживает язык HTML?
  3. Как передать информацию из одного фрейма в другой?
  4. Какие существуют атрибуты тэга «frame»?
  5. Какие элементы называются блочными (block-level) и текстовыми (text-level) элементами?
  6. Какие типы URL используются в ссылках на рисунки и другие Web страницы?
  7. Чем будет отличаться текст в броузере для следующих двух команд исходного текста:
    1. «a name="park"»«/a»Прогулка по парку
    2. «a name="park"»Прогулка по парку«/a»
